## PAI v0.1.0-rc1

First public pre-release of PAI (Pocket AI) — a bootable live-USB operating system for private local AI.

**Download and flash to a USB drive, then boot from USB on any x86_64 PC.**

### What's included

- Debian 12 (Bookworm) base, Sway desktop (Wayland)
- Ollama for local LLM inference
- Open WebUI at `localhost:8080`
- Firefox ESR
- UFW firewall (default-deny), MAC randomization, opt-in Tor
- Electrum Bitcoin wallet, aisleriot games
- Amnesic: RAM-only, leaves no trace on the host machine

### Flash to USB

**Linux / macOS:**
```bash
sudo dd if=pai-0.1.0-rc1-amd64.iso of=/dev/sdX bs=4M status=progress conv=fsync
```
Replace `/dev/sdX` with your USB drive (use `lsblk` to identify).

**Windows:** Use [Rufus](https://rufus.ie) or [balenaEtcher](https://etcher.balena.io).

### Boot

Plug in the USB, restart, press **F12 / F2 / DEL / Esc** (depends on your BIOS) and select the USB device. The desktop loads in about 60 seconds.

### Known issues in rc1

- Sway autostart requires `WLR_NO_HARDWARE_CURSORS=1` on some VMs — set it manually in the Sway config if the cursor doesn't appear. **Fixed in v0.1.0.**
- Open WebUI does not yet show the PAI logo branding. **Fixed in v0.1.0.**
- ARM64 build is experimental (Docker `--platform=linux/arm64` required).
