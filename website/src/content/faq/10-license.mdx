Yes — PAI's system integration code is released under the **GNU General Public License v3.0** (GPL-3.0).

You are free to:
- Use PAI for any purpose, including commercial
- Study how PAI works and modify it
- Redistribute PAI to others
- Distribute your modified versions

The only condition: if you distribute a modified version, you must make the source code available under the same GPL-3.0 license.

**Included software retains its own licenses:**
- Ollama: MIT license
- PAI UI (derived from LobeChat): MIT license
- Debian packages: various (GPL, LGPL, BSD, MIT) — all DFSG-compliant
- Firefox ESR: Mozilla Public License 2.0

**In practice:** you can fork PAI, add your own tools, and distribute it — as long as you publish your fork's source code. The GPL ensures PAI remains open and improvements flow back to the community.

See the [LICENSE](https://github.com/nirholas/pai/blob/main/LICENSE) file for the full text.
