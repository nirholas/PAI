question: "Why not just use Tails?"
order: 2
---

Tails is excellent for anonymous browsing and communication. PAI solves a different problem: **private local AI**.

| | Tails | PAI |
|---|---|---|
| Primary purpose | Anonymous browsing | Private local AI |
| Tor integration | Always-on (everything) | Opt-in |
| AI capabilities | None | Full local LLM via Ollama |
| Desktop | GNOME | Sway (lighter) |
| Amnesic | Yes | Yes |
| ISO size | ~1.9 GB | ~912 MB |

If your goal is anonymous web browsing and encrypted communication, use Tails. If your goal is conversing with an AI model without sending your prompts to a corporation, use PAI.

Many users who need both run PAI with Tor enabled — you get PAI's local AI + Tails-style network anonymity in one system.
